# Zendesk
Zendesk Scripts and Tools

<b>DC Manager</b>: Tool that automatically creates the dynamic content of a Zendesk instance.

<b>Migration Tool</b>: This tool makes it easier to migrate custom field configurations from one Zendesk instance to another.

<b>Delete Tool</b>: Tool to help delete custom fields and dynamic content in a Zendesk instance.

<b>Custom Object Manager</b>: A custom app that allows you to add, update, and edit Zendesk custom object records.

<b>Advanced Viewer</b>: A custom app replacement for Zendesk Support's native Views.

<b>API Ticket Tool</b>: Tool to create tickets using the Zendesk API.
