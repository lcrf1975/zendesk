# Zendesk
Zendesk Scripts and Tools

<b>DC Manager</b>: A tool that automatically creates the dynamic content of a Zendesk instance.

<b>Migration Tool</b>: This tool makes it easier to migrate custom field configurations from one Zendesk instance to another.

<b>Delete Tool</b>: A tool to help delete custom fields and dynamic content in a Zendesk instance.

<b>Custom Object Manager</b>: A custom app that allows you to add, update, and edit Zendesk custom object records.
